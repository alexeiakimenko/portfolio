1.	Выведите перечень сумм заказов продавца Rifkin.
SELECT o.amt
FROM Salespeople s,Orders o
WHERE s.snum=o.snum AND s.sname='Rifkin'

SELECT amt
From Orders
WHERE snum=
    (SELECT snum
    FROM Salespeople
    WHERE sname LIKE 'Rifkin'
    )

2.	Определите  комиссионные всех продавцов, обслуживающих заказчиков в Риме
SELECT com
FROM Salespeople
WHERE snum IN
    (SELECT snum
    FROM Customers
    WHERE city LIKE 'Rome'
    )

SELECT s.com
FROM Salespeople s,Customers c 
WHERE s.snum=c.snum AND c.city LIKE 'Rome'


3.	Напишите запрос, который определит заказчиков имеющих рейтинг ниже среднего.
SELECT rating
FROM Customers
WHERE rating<
    (
    SELECT AVG(rating)
    FROM Customers
    )
